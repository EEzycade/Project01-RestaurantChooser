# Restaurant Recommender App

## Purpose

AS A USER who does not know where to go for dinner
I WANT TO enter my address and preferred travel distance
SO THAT I can view recommendations for where to eat

## Built With

- HTML
- CSS
- Javascript
- Cirrus
- Documenu API
- positionstack API
